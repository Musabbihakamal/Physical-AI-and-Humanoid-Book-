# Quiz Creator Agent

The Quiz Creator Agent generates MCQs, short answer questions, and coding exercises with configurable difficulty based on user profile.

## Purpose
- Create assessment materials for book chapters
- Generate different types of questions (MCQ, short answer, coding exercises)
- Adjust difficulty based on user profile and preferences
- Test student understanding of chapter content

## How It Works
1. Analyzes chapter content for key concepts and topics
2. Generates various question types based on content
3. Adjusts difficulty based on user profile (experience level, learning goals)
4. Creates answer keys and explanations
5. Formats output as Docusaurus-compatible Markdown

## Usage
The Quiz Creator can be accessed through the agent interface in the book content pages. Select a chapter and specify question types and difficulty preferences.

## Input Requirements
- Chapter content in Markdown format
- User profile with experience level and preferences
- Desired question types and difficulty settings

## Output Format
- Multiple choice questions with answer options
- Short answer questions with expected answers
- Coding exercises with requirements and examples
- Difficulty-appropriate questions based on user profile
- Docusaurus-compatible formatting
