# Chapter Generator Agent

The Chapter Generator Agent creates well-structured educational content that follows pedagogical best practices based on module focus.

## Purpose
- Generate complete book chapters automatically
- Create content that follows educational best practices
- Structure content with proper headings and organization
- Include code blocks, diagrams, and exercises as appropriate

## How It Works
1. Takes a module focus or topic as input
2. Generates chapter structure with appropriate headings
3. Creates content that follows pedagogical best practices
4. Includes code examples, diagrams, and exercises where appropriate
5. Formats output as Docusaurus-compatible Markdown

## Usage
The Chapter Generator can be accessed through the agent interface. Provide a module focus or topic to generate a complete chapter.

## Input Requirements
- Module focus or topic description
- Optional: Target audience and difficulty level
- Optional: Specific requirements or constraints

## Output Format
- Well-structured chapter with headings and subheadings
- Educational content appropriate for the topic
- Code blocks and examples where relevant
- Diagrams and exercises as appropriate
- Docusaurus-compatible formatting
