# Code Explainer Agent

The Code Explainer Agent generates detailed explanations for code examples with specific highlighting of ROS 2 and Isaac Sim commands.

## Purpose
- Explain complex code examples in book chapters
- Highlight ROS 2 and Isaac Sim specific commands
- Provide context for code snippets
- Enhance understanding of technical implementations

## How It Works
1. Parses code snippets to identify language and framework
2. Detects ROS 2 and Isaac Sim specific commands and functions
3. Generates detailed explanations for each code block
4. Highlights important commands for quick reference
5. Formats output as Docusaurus-compatible Markdown

## Usage
The Code Explainer can be accessed through the agent interface in the book content pages. Select a code block or provide code content for explanation.

## Input Requirements
- Code content with clear language identification
- Optional: User profile for personalized explanation depth

## Output Format
- Detailed explanations for code blocks
- Highlighted ROS 2 and Isaac Sim commands
- Contextual information about code functionality
- Docusaurus-compatible formatting
