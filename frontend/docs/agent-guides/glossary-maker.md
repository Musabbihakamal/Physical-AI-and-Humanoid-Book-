# Glossary Maker Agent

The Glossary Maker Agent automatically generates glossaries from chapter content with links to their occurrences in the text.

## Purpose
- Extract important technical terms from book chapters
- Provide clear definitions for complex terminology
- Link terms back to their occurrences in the content
- Enhance learning by providing quick access to definitions

## How It Works
1. Analyzes chapter content for technical terms and concepts
2. Generates clear, concise definitions for identified terms
3. Creates links from glossary entries back to their occurrences in the text
4. Formats output as Docusaurus-compatible Markdown

## Usage
The Glossary Maker can be accessed through the agent interface in the book content pages. Simply select a chapter and request glossary generation.

## Input Requirements
- Chapter content in Markdown format
- Optional: User profile for personalized difficulty level

## Output Format
- A glossary with terms and definitions
- Hyperlinks from terms back to their occurrences in the text
- Docusaurus-compatible formatting
