# Subagent System Documentation

This section provides comprehensive documentation for all subagents in the system. Each agent serves a specific purpose in enhancing the educational experience of the book.

## Available Agents

- **[Glossary Maker](./glossary-maker.md)**: Automatically generates glossaries from chapter content with links to their occurrences.
- **[Code Explainer](./code-explainer.md)**: Generates detailed explanations for code examples with specific highlighting of ROS 2 and Isaac Sim commands.
- **[Quiz Creator](./quiz-creator.md)**: Generates MCQs, short answer questions, and coding exercises with configurable difficulty based on user profile.
- **[Chapter Generator](./chapter-generator.md)**: Creates well-structured educational content that follows pedagogical best practices based on module focus.
- **[Book Content Writer](./book-content-writer.md)**: Helps create and refine educational content for book chapters with a focus on clarity and pedagogical effectiveness.

Each agent can be used independently or in combination with others to enhance the learning experience.
